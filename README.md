<p>-Language: PHP (LARAVEL FRAMEWORK)</p>
<p>-Libraries used: JQUERY and AJAX Code follow the MVC model.</p>
<p>-The project consists of 2 parts: Client (client side) and Admin (Admin side).</p>
<p>-There is deep decentralization functionality</p>
<p>•Admin:</p>
<p>-Login</p>
<p>-CRUD: Pages, posts and categories posts, products and product categories, sliders, admins, permissions, photos,...</p>
<p>-RUD: Orders, customers</p>
<p>-Filter by status, search by name: all modules</p>
<p>-View sales, number of products sold, orders by status, view order details</p>
